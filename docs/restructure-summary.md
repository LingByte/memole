## 最终项目结构

```
memmole/
├── main.go                    # 程序入口
├── go.mod                     # Go模块定义
├── docs/                      # 📚 项目文档
│   └── architecture.md        # 架构设计文档
├── pkg/                       # 📦 公共核心模块
│   ├── ast/                   # 抽象语法树
│   ├── object/                # 对象系统
│   ├── lexer/                 # 词法分析器
│   ├── parser/                # 语法解析器
│   ├── evaluator/             # 求值器
│   ├── logger/                # 日志系统
│   ├── config/                # 配置管理
│   └── storage/               # 存储系统
├── builtins/                  # 🔧 内置函数
├── network/                   # 🌐 网络模块
├── database/                  # 🗄️ 数据库模块
├── repl/                      # 💻 REPL环境
├── examples/                  # 📝 示例代码
└── *.mml                      # 🧪 测试文件
```
